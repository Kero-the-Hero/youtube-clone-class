#1.3 What is Node JS
 - 크롬 V8 자바스크립트 엔진으로 빌드된 자바스크립트 런타임
  이게 뭔 소리임?
   -> 브라우저 밖에서 돌아가는 자바스크립트임
   -> 초창기엔 자바스크립트가 웹에서만 쓰일수 있었는데
   -> Ryan씨가 자바스크립트를 너무 사랑한 나머지 웹 말고 다른 곳에서도 사용하고 싶어서
      Node JS를 만들어 냄

#1.4 What is NPM
 - 자바스크립트 언어를 위한 패키지 매니저
 - NPM은 node JS와 함께 쓰여야 한다는 뜻
 - 이미 만들어진 패키지를 가지고 쉽게 개발에 착수 할 수 있다
 - 패키지들을 검색하는 건 npmjs.com에 가서 해보자
